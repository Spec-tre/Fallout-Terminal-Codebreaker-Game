# Fallout-Terminal-Codebreaker-Game
A codebreaker game with the ROBCO (TM) boot sequence.

The game rules:

1. The program will randomly generate a set digit code with given numbers.

2. The program will give hints to how close you are to cracking the code.

3. These hints are based upon correct number and position or a misplaced correct number.

4. You are given a certain level of Attempts to crack the code.

This is a first project.
